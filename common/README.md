## Requirement
- [node.js v5.0以降](https://nodejs.org/en/)
- [gulp](http://gulpjs.com/)


## Usage

1. 各モジュールのインストール
>npm install

( もし、typingsが作成されなかったら、```npm run install```コマンドを打ってください。 )

2. 起動
>npm run start

3. ファイルの監視をしてbrowser-syncで起動する
> npm run sync
