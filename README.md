# Particle Develop

![Particle Develop](https://ics.media/wp-content/uploads/2016/01/160125_particle_develop.png)

## Description

Particle DevelopはHTML5製のデザインツールです。雪や炎、キラキラなどのパーティクル表現を作成でき、ベクター画像として保存することができます。Adobe Illustratorのイラスト制作にも役立ちます。
